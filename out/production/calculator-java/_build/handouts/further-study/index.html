Things to notice in Calculator.java

  • .equals() to compare strings in if-statement

  • .split() works similarly to Python

  • Casting a string to a float works like Float.parseFloat(someString)

  • The way the import statements work for Arithmetic

  • Getting length of array is tokens.length

  • Indexing into arrays is the same
